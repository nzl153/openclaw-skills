# Novel Writer Skill

一个面向番茄小说平台的中文网文写作辅助 Skill，为 OpenClaw Agent 提供专业的网文创作能力。

## 功能特点

- 📝 **小说创作** — 支持穿越/重生、武侠、都市、系统流、游戏电竞、科幻末世等主流题材
- 💾 **存档记忆** — 每章写完后自动记录剧情进度、角色状态、世界观进展，下次会话无缝衔接
- 📖 **写作指南** — 内置番茄平台攻略（算法机制、字数节点、首秀策略）+ 各题材开局技巧 + 爽点设计 + 断章法
- 🎯 **番茄适配** — 针对番茄小说读者阅读习惯优化：快节奏、高爽点、强钩子
- 🔄 **风格定制** — 可针对不同作者的文风、节奏、金手指类型做个性化调整
- 🌐 **多题材覆盖** — 内置六大主流题材的写作指南，不用切换 Skill

## 使用方法

### 第一步：安装

```bash
clawhub install novel-writer
```

或手动下载后解压到 `skills/` 目录。

### 第二步：开始写小说

告诉 Agent 你想写小说，Agent 会自动加载存档（如果有的话）或引导你创建新书设定。

### 第三步：存档管理

每次写作会话结束时，Agent 会自动更新存档文件 `references/<书名>-存档.md`，记录：

- 最后写到的章节和场景
- 角色当前状态
- 世界观展开进度
- 待处理的伏笔
- 下一章的方向建议

## Skill 结构

```
novel-writer/
├── SKILL.md                              # 主技能文件
├── references/
│   ├── _template/                        # 📁 通用存档模板
│   │   ├── 0-设定速查.md                 #   世界观/金手指规则
│   │   ├── 1-角色档案.md                 #   角色表+实力变化
│   │   ├── 2-剧情进度.md                 #   当前进度+伏笔清单
│   │   ├── 3-章节摘要.md                 #   每章一行速查
│   │   ├── 4-时间线.md                   #   故事内天数
│   │   └── 5-已纠正错误.md               #   ⚠️ 写作前必读
│   └── writing-guide.md                  # 网文写作指南（各题材攻略+爽点+断章）
└── scripts/
    └── save-novel-state.py               # 存档管理脚本
```

## 存档系统（长篇小说专用）

几十万字的长篇小说，单文件存不下。每本书在 `references/` 下创建独立目录：

```
references/
  《书名》/
    ├── 0-设定速查.md          # 世界观/力量体系（基本不变）
    ├── 1-角色档案.md          # 角色表+实力变化记录
    ├── 2-剧情进度.md          # 当前章节计划+未解伏笔
    ├── 3-章节摘要.md          # 每章一行速查+结尾场景
    ├── 4-时间线.md            # 小说内天数记录
    └── 5-已纠正错误.md        # ⚠️ 写作前必读
```

写作前按层级顺序读取：
1. **必读少量**: 5→0→2（错误→设定→进度）
2. **必读略多**: 1→3（角色→近3章结尾）
3. **按需查阅**: 4（时间线）

新建书时直接复制 `_template/` 目录：
```bash
cp -r references/_template/ references/《你的书名》/
```

## 适用场景

- 在番茄小说等平台连载网文
- 需要 Agent 帮忙写初稿/润色/续写
- 开新书时需要设定存档
- 几十万字长篇需要保持剧情连贯
- 多人协作写书时需要剧情连贯
- 收全勤奖需要日更4000字

## 支持的题材

当前内置以下题材的写作攻略：

| 题材 | 特点 |
|------|------|
| 穿越/重生修仙玄幻 | 修炼体系、越级打怪、经典网文范式 |
| 武侠 | 低武中武、内力招式、江湖门派 |
| 都市现代 | 生活代入感、职业装逼、财富逆袭 |
| 系统流/签到流 | 系统规则、任务设计、金手指成长 |
| 游戏/电竞 | 操作描写、职业体系、战队组建 |
| 科幻/末世 | 世界观建立、生存压力、异能/黑科技 |
