# 番茄小说网文写作指南

> 整理自多位资深网文作者的公开经验，适用于番茄小说平台各主流题材

## 一、番茄平台核心逻辑

### 算法关键数据
- **追读率 > 完读率** — 追读率决定生死，完读率影响收入
- **14天给量周期** — 前6天尝试给量→精准给量，后8天动态给量
- **静默期** — 15~16万字进入第一个静默期，20万字重新来量
- **评分机制** — 评分=吸量+留存+完读率综合判断，与读者打分无关

### 字数节点
- **3万字** — 书荒（可发推书）
- **8万字** — 验证期+7天首秀
- **20万字** — 书测（多书名实验）
- **日更4000字** — 全勤奖门槛

### 写作节奏
- **每章2000字** — 番茄推荐节奏，不要少于1500
- **前100字定生死** — 番茄读者滑屏阅读，开头没钩子直接划走
- **前3章决定留存** — 必须让读者在前三章建立追读欲望

## 二、各题材开局要点

### 二-A、穿越/重生类（修仙/玄幻/异界）

#### 第一章必须做到的
1. **身份立住** — 主角穿越前的身份、处境、性格，用生活细节塑造代入感
2. **处境压力** — 穿越后立刻面临的现实问题（危险、饥饿、生存危机）
3. **环境渲染** — 不靠旁白说明，靠五感描写（气味、声音、触感、光线）
4. **金手指不要急着出** — 让读者先关心主角的处境，再给解决方案

#### 开局节奏控制
```
0~500字:   醒来→感知环境→发现不对劲→确认穿越（渲染陌生感）
500~1500字: 独自探索→遇到困难/危机→靠自身能力初步应对
1500~2500字: 危机升级/遇到关键角色→金手指激活
2500~3500字: 借助金手指初步解决问题→引出后续目标+断章
```

#### 黄金公式
> **低起点 + 金手指 + 周围人不看好 + 读者期待 → 出手打脸 → 收获**

#### 特别提醒
- **身体穿越 vs 灵魂穿越** — 不同穿越方式影响后续剧情展开方式
- **原作主角处理** — 如果穿越到游戏/小说世界，原作主角可以是好哥们也可以是竞争者
- **开发者/创作者穿越** — 主角知道自己世界的规则但不知道后续发展，天然悬念

---

### 二-B、武侠类（低武/中武）

#### 与修仙/玄幻的区别
- **体系更稳定** — 内力+招式+轻功+兵器，不容易战力崩坏
- **天花板明确** — 读者天然理解江湖的层级，不需要大段说明
- **门派势力** — 武林门派、正邪对立、江湖规矩，天然剧情驱动力
- **兵器/武功秘籍** — 可以成为章节钩子和爽点载体

#### 第一章要点
1. **江湖氛围** — 酒馆、铁匠铺、镖局、比武擂台，用场景塑造江湖感
2. **主角处境** — 可以是初入江湖的菜鸟、被灭门的遗孤、隐姓埋名的高手
3. **武功设定自然带出** — 通过打斗/对话展示内力境界，不写说明文
4. **武林世界观** — 第一章就要让读者知道这个世界的江湖大概什么样

#### 常见爽点
- **越级反杀** — 靠技巧/经验战胜比自己内力深厚的人
- **秘籍奇遇** — 捡到/得到失传的武功秘籍
- **身份暴露** — 隐藏的高手身份在关键时刻被揭穿
- **以武会友** — 不打不相识，结交江湖好友

---

### 二-C、都市/现代题材

#### 第一章必须做到的
1. **生活感立住** — 主角的日常生活细节要真实（上班、租房、挤地铁、还贷），让读者产生代入感
2. **现实困境** — 缺钱、被欺负、职业瓶颈，先刻画一个读者能共情的处境
3. **金手指契合生活** — 金手指要和主角的现实生活产生化学反应，不是为了爽而爽
4. **反差感** — 获得金手指后低调发育，让读者期待主角在熟人面前展现实力的时刻

#### 常见爽点套路
- **职业装逼** — 医生/律师/程序员等专业领域展现超强能力
- **财富逆袭** — 从穷到富的过程要写足细节（买房、买车、被看不起的人刮目相看）
- **情感反转** — 前女友看不起→主角成功→前女友后悔（但要克制，不要过度狗血）
- **隐藏身份** — 主角表面普通，实则是大佬，在关键时刻暴露

---

### 二-D、系统流/签到流

#### 核心公式
> **开局得系统 + 签到/做任务拿奖励 + 日常装逼 + 大事件爆发 + 系统升级**

#### 第一章必须做到的
1. **系统规则清晰** — 第一章就要让读者明白系统怎么用（签到、任务、抽奖？）
2. **首奖要香** — 第一个奖励必须让读者觉得"这系统牛批"，建立期待
3. **系统有成长性** — 后续系统会升级/解锁新功能，保持新鲜感
4. **避免无脑签到** — 系统只是工具，主角的成长和选择才是看点

#### 注意事项
- 系统面板可以偶尔出现，不要每章都贴满面板
- 系统任务要推动剧情，不是让主角原地打转
- 系统可以有自己的性格/来历，中期可以揭露系统背后的故事

---

### 二-E、游戏/电竞题材

#### 第一章要点
1. **游戏世界建立** — 让读者快速理解是什么游戏（MMO/FPS/MOBA？）
2. **主角定位** — 可以是职业选手退役、路人王、游戏策划、代练等
3. **专业感** — 操作描写要有真实感，懂游戏的读者一眼就能看出水平
4. **现实与游戏的平衡** — 游戏里的成就影响现实生活（收入、地位、人际关系）

#### 常见爽点
- **神操打脸** — 所有人都觉得必输的局面，靠神级操作翻盘
- **身份揭穿** — 小号/马甲被认出是传说中的大神
- **组建战队** — 从零开始召集队友，团队成长的爽感
- **现实装逼** — 游戏里的技术在现实中有用（手速、反应、策略思维）

---

### 二-F、科幻/末世题材

#### 第一章要点
1. **世界观快速建立** — 末世/科幻背景要用第一章就让读者明白"这是一个什么样的世界"
2. **生存压力** — 资源稀缺、危险无处不在，紧迫感拉满
3. **金手指要契合题材** — 异能觉醒、系统辅助、前世记忆、黑科技等
4. **人性刻画** — 末世文的核心魅力在于极端环境下的人性抉择

#### 爽点方向
- **异能觉醒** — 觉醒稀有/强大能力，碾压普通人
- **基地建设** — 从零开始建基地，收集资源、招募人手
- **打脸圣母/反派** — 末世里圣母心泛滥的角色活不过三章
- **科技碾压** — 用前世的科技知识碾压当前时代

---

## 三、爽点设计

### 爽点层级
- **小爽点** — 每章至少1个（打脸、升级、收获、反转）
- **中爽点** — 每10章一个小循环（拉仇恨1~2章→被压1章→反踩1章→打脸2~3章→收获取胜）
- **大高潮** — 每30万字一卷（两小高潮+一大高潮）

### 爽点核心原则
1. **高潮一定要有观众** — 最好是和主角有关系的人，通过他们的反应放大爽感
2. **收获才是核心** — 打脸不是目的，打完脸后得到什么才是读者真正想看的
3. **好处要用出去** — 法宝、装备等尽快用掉，别积压
4. **越级打怪多样化** — 打比自己高级的 / 打两个同级的 / 打地位高的 / 打有环境优势的

### 装逼的艺术
- **反向装逼** — 主角低调行事，读者知道但书中角色不知道，期待感拉满
- **差一点点就赢了** — 所有人都觉得你差一点赢了（其实你还差很多），观众为你可惜，下次你真正赢了，爽感翻倍

---

## 四、角色塑造

### 主角
- **和读者平等的起点** — 开局给一个比读者更低或平等的位置，天然代入感
- **可以强大但不能顺心** — 一旦主角万事如意，书就没有吸引力了
- **主角的处境生活细节** — 要有家人朋友，有要面对的现实问题
- **主角越惨，读者期待越高** — 惨是为了打脸准备的，但一定要快速完成升级再打脸

### 反派分级
| 级别 | 特点 | 处理方式 |
|------|------|----------|
| 炮灰级 | 自以为是、目中无人 | 快速打脸，1~2章解决 |
| 精英级 | 有实力、自命不凡 | 铺垫再打，3~5章 |
| 小BOSS | 颠倒黑白、徇私枉法 | 多条线推进，10章左右 |
| 大BOSS | 邪恶化身、幕后黑手 | 全书的终极对手 |

### 金手指/系统角色
- **前期** — 纯粹功能性存在，提供信息和帮助
- **中期** — 偶然流露出类人反应，开始有情感萌芽
- **后期** — 完整的人格觉醒，成为独立角色
- **注意** — 不要过早暴露金手指的全部能力，留成长空间

---

## 五、断章技巧

### 黄金断章法
- **突破前断** — 主角即将突破/升级时断
- **大战前断** — 大战一触即发时断
- **秘密揭晓前断** — 即将揭露关键信息时断
- **反转前断** — 主角处境反转的临界点断
- **死亡威胁前断** — 主角面临生死危机时断（但要保证下章能解决）

### 章节内部结构
```
【钩子开篇】→ 前100字制造悬念/冲突/爽点
【中段推进】→ 推进剧情+穿插爽点+展示金手指
【结尾断章】→ 留钩子（让读者想看下一章）
```

## 六、文笔与描写

### 番茄风格特点
- **段落短** — 2~4句一段，不要大段说明文
- **对话多** — 对话推进剧情比描写快
- **小白文精髓** — 不关键的地方用说明文快速带过，关键地方才展开写
- **不用太在意细节违和** — 小读者不会在意局部细节的违和，整体大方向把握好就行

### 情绪调控
- **主角得到好处** → 用严肃、正式的词句写
- **主角损失什么** → 用欢快、幽默的词句写（框架效应）
- **一个情节走完** → 主角得到的比损失的多两倍以上

---

## 七、世界观铺设

### 通用原则
- **不要写设定文档** — 通过剧情和对话自然展示
- **等级/力量体系要森严** — 等级差距、优越感、危机感都要有
- **随主角探索自然展开** — 读者和主角一起发现世界

### 不同题材的展开方式
| 题材 | 世界观展开方式 |
|------|---------------|
| 修仙/玄幻 | 修炼境界在日常打斗和突破中自然展示 |
| 武侠 | 通过对话、说书、门派介绍展示江湖势力 |
| 都市 | 通过主角的职业/社交圈展示社会规则 |
| 系统流 | 系统面板和任务说明本身就是世界观载体 |
| 末世/科幻 | 通过环境描写、幸存者对话、废弃设施展示 |

---

## 八、长期连载策略

### 大纲的重要性
- **主线是脊椎** — 主角到底要干嘛？什么驱动他不断行动？
- **细纲必须有** — 人物出现、消失，事件开始、结束，都要事先安排
- **30万字出一卷** — 形成两个小高潮+一大高潮的节奏

### 卡文应对
- 找不长眼的角色来杀一下
- 填前面埋的坑
- 写人物互动（装逼在互动中完成，红花需要绿叶配）
- 情绪低落时，写主角突破/开挂

### 防止写崩
- **多短线，少长线** — 不要让仇人过夜，最多一万字内要有进展
- **之前定下的设定不能轻易推翻**
- **每一场战斗的招式、过程、结尾都想好再动笔**
- **120万字内完本** — 不要强行拖字数，保持节奏紧凑
- **结尾预留空间** — 完本前让主要角色都有交代，不留明显坑
